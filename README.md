# 🌊 北京市水质监测数据看板

数据来源：[北京市生态环境局](https://sthjj.beijing.gov.cn/)

## 在线查看

👉 **[点击查看完整交互式看板](https://hjing-nobody.github.io/beijing-water-quality/)**

如果上方链接无法打开，请直接点击下方文件列表中的 `index.html` 文件查看完整看板。

## 数据说明

- 监测断面覆盖：河流、湖泊、水库
- 水质类别：Ⅰ~Ⅴ类及劣Ⅴ类
- 每月15日、25日自动更新

## 数据格式

| 文件 | 说明 |
|------|------|
| `index.html` | 交互式看板（点击即可在 GitHub 中渲染查看） |
| `北京水质数据.xlsx` | Excel 数据表 |
| `北京水质数据.json` | JSON 格式数据 |
